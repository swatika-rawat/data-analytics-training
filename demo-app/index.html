Welcome to Data Analytics through Elastic Stack




In order to access data directly from Elasticsearch please add the following in elasticsearch.yml file located under config directory of Elasticsearch installation:

http.cors.enabled: true
http.cors.allow-origin: "*"
In a real application we should never expose our datastore and all request should go through application server.
